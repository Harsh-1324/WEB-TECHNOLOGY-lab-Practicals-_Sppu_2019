# WEB-TECHNOLOGY-lab-Practicals-_Sppu_2019
Comprehensive Web Technology laboratory practicals, assignments, and mini-projects using HTML, CSS, JavaScript, Servlets, JSP, PHP, and modern frameworks based on Savitribai Phule Pune University (SPPU) 2019 Computer Engineering curriculum.

This repository helps students learn and implement client-side and server-side technologies to build dynamic and responsive web applications.

📘 Course Information
University: Savitribai Phule Pune University (SPPU)
Course: 310257 – Web Technology Laboratory
Pattern: 2019 Course
Branch: Third Year Computer Engineering
